# FixedIT — Demo Site

A concept landing page for **FixedIT Consulting AB** (fixedit.ai), built by Zaitex.
Edge-native ACAP platform for Axis devices — Observer · Plumber · Builder.

## Stack
- Vite + React 18
- Tailwind CSS v3
- iconify-icon, lucide-react

## Develop
```bash
npm install
npm run dev      # http://localhost:5173
```

## Build
```bash
npm run build    # outputs to dist/
npm run preview
```

## Deploy (Vercel)
- Framework preset: **Vite**
- Build command: `npm run build`
- Output directory: `dist`
